## 프로젝트 소개

**"지금 당장 갈 수 있는 카페를 찾고 싶은데, 영업시간을 일일이 확인하는 게 번거롭다"**

이러한 문제를 해결하기 위해 시작된 프로젝트입니다. 기존 지도 앱들은 단순히 가까운 카페를 보여주지만, **지금 이 시간에 영업 중인지**를 바로 알 수 없습니다.
